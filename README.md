**ENVX** is a revolutionary desktop application that sets up complete development environments with a single click. Just paste any GitHub URL, and ENVX automatically clones, configures, and runs the project - regardless of the technology stack.

![ENVX Desktop Demo](screenshot.png)

## ✨ Features

### 🎯 **One-Click Setup**
- Paste any GitHub URL → Click SETUP → Done!
- Automatically clones repositories to your chosen location
- Works with **ANY** project type

### 🧠 **Smart Language Detection**
- **Node.js** (npm start, next dev)
- **Python** (Django, Flask, FastAPI)
- **PHP** (built-in server)
- **Go**, **Rust**, **Ruby**, and more!

### 🔧 **Auto-Healing**
- Detects and fixes broken dependencies
- Installs missing packages automatically
- Handles legacy projects with outdated requirements

### 🗄️ **Database Setup**
- MySQL auto-installation & configuration
- PostgreSQL setup
- Redis cache configuration
- Database import from schema files

### ⏱️ **Time Travel** (CLI)
- Environment snapshots every 5 minutes
- Jump back to any point in time
- "It worked yesterday" → Problem solved!

### 🧬 **DNA Fingerprinting**
- Creates unique DNA for every environment
- Share DNA codes with teammates
- Recreate exact environments anywhere

### 🖥️ **Desktop App Features**
- 📁 **Choose clone location** (Desktop, Documents, etc.)
- 🚀 **One-click VS Code** integration
- 🌐 **Auto-opens browser** when server starts
- 📊 **Real-time terminal output**
- 🎨 **Beautiful, modern UI**
